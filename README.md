# Site de Casamento - Maria & Ulisses

Galeria de fotos interativa com upload, visualização em lightbox e exclusão com senha.

## Requisitos
- Python 3.6+
- Flask

## Instalação
1. Clone o repositório
2. Instale as dependências: `pip install flask`
3. Execute: `python app.py`
4. Acesse: `http://localhost:5000`

## Senha padrão para excluir
`socorro123` (altere no código)

## Personalização
- Nomes, data e mensagens estão no template HTML dentro de `app.py`.
